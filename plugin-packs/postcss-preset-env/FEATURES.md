-------

Manually enable a feature :

```js
postcssPresetEnv({
	features: {
		'<ID>': true
	}
})
```

Or disable one :

```js
postcssPresetEnv({
	features: {
		'<ID>': false
	}
})
```

Set feature options :

```js
postcssPresetEnv({
	features: {
		'<ID>': {
			preserve: false
		}
	}
})
```
